# Pugmill Community — Recipe

A Pugmill CMS recipe that adds a public recipe registry to your site. Community members sign in with GitHub to browse, submit, and manage plugins, themes, workflows, and PNA cartridges.

**Requires:** Pugmill CMS v0.2.0+

See [RECIPE.md](./RECIPE.md) for installation instructions.

---

## What's included

- Recipe registry at `/recipes` with type filters and pagination
- GitHub OAuth sign-in (separate from the Pugmill admin session)
- Recipe submission, indexing from GitHub, download tracking
- Member profiles with tier and contribution score
- Admin plugin page for OAuth status and navigation setup
